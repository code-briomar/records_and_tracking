Welcome to the Records & Tracking System

This guide will help you get started and test the main features of the system.

1. Logging In

  1. Open the application on your computer.
  2. Enter your username and password on the login screen.
  3. Click Login.

2. Adding a New Case File

  1. Go to the Files or Case Files section from the main menu.
  2. Click Add New File or Add New Case.
  3. Fill in the required details (case number, type, purpose, etc.).
  4. Click Save or Submit.
  5. You should see the new file appear in the list.

3. Searching for Files

  1. Use the Search bar at the top of the Files section.
  2. Type a case number, type, or keyword and press Enter.
  3. Matching files will be displayed below.

4. Syncing Data

  1. Ensure you are connected to the internet.
  2. Click the Sync button (if available) or wait for automatic sync.
  3. Check the status messages for successful sync with Supabase.

5. Viewing Audit Logs

  1. Go to the Audit Logs section from the menu.
  2. Review recent actions and changes made in the system.
Tip: If you encounter any issues, check your internet connection and ensure your login credentials are correct. For further help, contact your system administrator.

6. Logging Out

  1. Click your profile or the menu icon.
  2. Select Logout to safely exit the system.

Testing Checklist

  • ✅ Can log in and log out
  • ✅ Can add a new file/case
  • ✅ Can search for files
  • ✅ Can see files sync to/from the cloud
  • ✅ Can view audit logs

Thank you for using the Records & Tracking System!
